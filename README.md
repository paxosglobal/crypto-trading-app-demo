# Crypto Trading App

A fast and modern Vite application for crypto trading.

## Setup

We suggest [nvm](https://github.com/nvm-sh/nvm/tree/master) to manage Node.js versions.

```sh
nvm install
nvm use
npm install
```

## Development

- `npm run dev` - Run the app in development mode
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build
